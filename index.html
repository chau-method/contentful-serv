API Routes

/api/cards/:lang

lang (optional)

Returns an Array

Grabs all the cards/tiles. Returns cards that can be promo, title, deals, products. Takes an optional lang param that is used to target langauge translations

/api/products/:lang

lang (optional)

Returns an Array

Returns all the products. Takes an optional lang param that is used to target langauge translations.

/api/group/product/:id/:lang

id (required) lang (optional)

Returns an Array

Returns all the products with the matching parentRef id. Takes an optional lang param that is used to target langauge translations.

/api/type/:type/:lang

type (required) lang (optional)

Returns an Array

Returns all of the content type of whatever is in the type param. Takes an optional lang param that is used to target langauge translations.

/api/single/:type/:id/:lang

type (required) id (required) lang (optional)

Returns an Array

Returns an array with a single object of type param with the matching id. Takes an optional lang param that is used to target langauge translations.

/api/:locale/languages

locale (optional)

Returns an Array

Returns an array with all the copy laid out in key value pairs. Takes an optional locale param that is used to target the specific language. Defaults to English.

/api/:locale/pullinview

locale (optional)

Returns an Array

Returns the key value pairs for the copy on the pull in view. This has it's own api call to update the state on kiosk without pulling down the entire language object. Takes an optional locale param that is used to target the specific language's layout. Defaults to English.

/api/:locale/categories/:lang

locale (optional) lang (optional)

Returns an Array

Returns an array of the categories used on the category landing view. Contains cards and products displayed on the category landing view after you select from the left hand nav. Takes an optional locale param that is used to target the specific language's layout and navigation items. Takes an optional lang param that is used to target the specific language translations. Both default to English.
